# Claims Forecasting

## Overview
This project focuses on **forecasting the number of insurance claims** using historical data.  

Link to dataset: https://www.kaggle.com/datasets/floser/french-motor-claims-datasets-fremtpl2freq

Two models were implemented and compared:
- **Poisson Regression** 
- **XGBoost**

---

## Methods and Tools
- **Language:** Python  
- **Libraries:** `pandas`, `numpy`, `matplotlib`, `scikit-learn`, `xgboost`,  `statsmodels`,  `seaborn`
- **Models:** Poisson Regression, XGBoost Regressor
